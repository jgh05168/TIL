SSAFY SW 역량테스트

Intermediate(IM) -> Advanced(A) -> Professional(B) -> Expert(C)


IM : 자격시험
- 조건문, 반복문만 사용해도 충분히 해결 가능



A형 : 두문제 중 한 문제 취득 시  (A+ : 두문제 중 두 문제 취득 시)
- trial & error를 해결할 수 있는지 판단
- 문해력이 가장 중요하다.

문제 유형 :
 - simulation 문제
 - 완전탐색(dfs / bfs / back tracking)


B형 : 자료구조와 알고리즘을 조합하여 문제최적화가 가능한지 판단(다양한 것들)
- 어떤 형태가 주어진 뒤 문제 형태 해결
- 창의성이 있어야 된다.
- 문제 해석에 꽤 많은 시간이 소요된다.

- 명세서 파악 능력이 충분해야 한다.
